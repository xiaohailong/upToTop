-------
使用方法
-------

1.   引入样式文件 index.css（可以自行编辑样式和文件路径）

2.   引入脚本文件 uptotop.js 或者 uptotop-anonymous.js
-------
*ps: uptotop 和 uptotop-anonymous的区别：
 
uptotop 直接引用到页面即可
 
uptotop-anonymous 还要 new 一个对象:new upToTop({speed:1.1,time:16.7})*;
